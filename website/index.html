§ 01

what it does

Six things SinWeave does that your editor probably doesn't.

Multi-file agents

Ask for a change across the repo. SinWeave reads, reasons, and writes coherent diffs you can apply or reject one hunk at a time.

Bring your own key

OpenAI, Anthropic, Gemini, OpenRouter, xAI, Mistral, Groq — plug in whatever you already pay for. Verify the key in-app before you start.

Local models, first-class

Ollama, LM Studio, vLLM, and OpenAI-compatible endpoints auto-detect. Your code never leaves your machine if you don't want it to.

Quick Edit (⌘K)

Select, invoke, describe. A focused inline editor that rewrites the selection without leaving your cursor. No modal dance.

Autocomplete that reads context

Tab completion backed by your model of choice, aware of the file, the project, and your recent edits. Switch providers without switching editors.

Runs on VS Code

Every extension, every keybinding, every setting you already have. SinWeave is a fork that preserves what works and replaces the parts that don't.

§ 02

how it feels

A short loop. Think. Select. Ask. Apply.

  1. 01

    Open a file, start a chat.

    Drop a file into the chat panel or select a region. SinWeave builds the context; you write the prompt.

  2. 02

    Pick a model. Or don't.

    Defaults work. If you're tuning for cost, latency, or quality, switch providers per-feature in Settings.

  3. 03

    Review the diff.

    Every change ships as a reviewable patch with hunks you can accept, reject, or edit inline. No stealth writes.

  4. 04

    Keep working.

    Git history, extensions, debuggers, LSPs — all of VS Code stays intact. SinWeave adds; it doesn't remove.

§ 04

questions

Is SinWeave free?

The editor is free and open-source (MIT). You pay whichever model provider you choose — or nothing at all, if you run local models with Ollama, LM Studio, or vLLM.

Does SinWeave send my code anywhere I don't know about?

No. API requests go directly from your editor to whichever provider you configured. There's no SinWeave telemetry server between you and the model. If you use only local models, nothing leaves your machine.

How is this different from Cursor or Windsurf?

SinWeave is open-source. You can read every line of the AI pipeline, fork it, change it, and ship it. You also control the keys; there's no required account, no metered plans, no mandatory cloud tier.

Will my VS Code extensions work?

Yes. SinWeave is a fork of VS Code — the extension model, keybindings, themes, and settings files are identical.

What about auto-updates?

Packaged releases ship with the same auto-update mechanism as VS Code, pointed at the SinWeave release feed. You can also disable updates and build from source.

Can I use this in a company?

Yes. MIT-licensed. Use it, fork it, redistribute it. The model you choose may have its own terms — check your provider.
